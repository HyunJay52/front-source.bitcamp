Step 1: Check whether the database is listening or not: Step 2: RUN the CMD (Command Prompt) in Administrator mode. Step 3: For checking the status of listener you have to run command “lsnrctl status”. Step 4: If the Status of the listener is failed or error out do bellow steps. Step 5: Run the command “lsnrctl start” to start the listener.
