====== table 관련 태그들 ======

table : 표의 시작
tr : 한 줄 생성
td : 한 칸 생성
th : 강조하고 싶은 줄에 td대신 th(글자가 칸의 가운데 정렬, 글자가 진하게)

- column/열 단위로 묶어서 처리 -
colgroup : 열 단위로 나누겠다는 표시 
col : 열의 개수 만큼 지정 가능, 같은 효과를 적용하려면 span사용
이미지로 배경처리를 하면, 그리드 형식으로
칸이 채워질때까지, 반복해서 채워진다

- Caption: table 제목 -
table안에서 위치는 상관없이, 실행하면 항상 상단에 나타난다 

- thead - 표의 상단
- tbody - 표의 내용
- tfoot - 표의 하단
... 적용해도 일단 눈에 보이지는 않는다
 
- 속성 -
width, height > px 단위(화면크기와 상관없이 크기 고정) or %(화면크기에 따라 조정됨)
table에 넣으면 table전체 폭/높이가 지정

css는 html에서 바로 적용 가능한데, 처음 테그 안에서 적용해야한다 - in Line
1st row 첫번째 칸 1st row 두번째 칸 1st row 세번째 칸 or #ff0000=red
2nd row 첫번째 칸 2st row 세번째 칸
3nd row 첫번째 칸 3st row 세번째 칸
